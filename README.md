# minimal-alarm-clock

A clean, dark-themed alarm clock that runs entirely in the browser — no installs, no dependencies.

![screenshot](https://raw.githubusercontent.com/syun3091/minimal-alarm-clock/main/screenshot.png)

## Features

- Live digital clock display
- Add / toggle / delete alarms
- Optional alarm label
- Audio beep via Web Audio API
- 5-minute snooze
- Persists alarms in `localStorage`

## Usage

Open `index.html` in any modern browser — no build step needed.

## Tech

Vanilla HTML · CSS · JavaScript (ES2020)
